## 1. Añadir el Bot

Haz clic en el botón para invitar a Caatffee a tu servidor:

[![Añadir Caatffee](https://img.shields.io/badge/➕_Añadir_Caatffee-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.com/oauth2/authorize?client_id=1475842620063485993&permissions=8&scope=bot%20applications.commands)

> ⚠️ **Necesitas ser administrador** del servidor para añadir el bot.

---

## 2. Permisos Necesarios

Caatffee solicita permiso de **Administrador** para funcionar correctamente.

| Permiso | ¿Para qué? |
|---------|-----------|
| Administrador | Gestionar canales, roles, miembros |
| Enviar mensajes | Responder comandos y enviar embeds |
| Insertar enlaces | Mostrar embeds con imágenes |
| Ver canal | Acceder a los canales del servidor |
| Gestionar servidor | Editar información del servidor |

> 🔒 **Nunca** utilizamos estos permisos para nada fuera de lo que tú ordenas con comandos.

---

## 3. Primer Comando

Una vez añadido el bot, escribe en cualquier canal:

```
/help
```

Recibirás una guía completa con todos los comandos disponibles.

---

## 4. Configuración Básica Recomendada

### 🔔 Configurar Logs
```
/set-logs canal:#tu-canal-de-logs
```
Registra todas las acciones del servidor en ese canal.

### 🌸 Configurar Bienvenida
```
/set-welcome activar canal:#bienvenidas mensaje:¡Bienvenido/a a {servidor}, {usuario}!
```

### 👋 Configurar Despedida
```
/set-goodbye activar canal:#despedidas mensaje:Hasta luego, {usuario} 👋
```

---

## 5. Panel Web

Puedes gestionar tu servidor directamente desde la web oficial:

🌐 [caatffee.github.io/Caatffee](https://caatffee.github.io/Caatffee/)

1. Haz clic en **"Iniciar sesión"** con tu cuenta de Discord
2. Ve a **"Ver mis servidores"**
3. Selecciona tu servidor
4. Configura bienvenidas, despedidas y envía mensajes desde ahí
