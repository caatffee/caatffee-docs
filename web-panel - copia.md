## What is the Web Panel?

Caatffee's web panel lets you manage your server **directly from your browser**, without needing to open Discord.

🌐 **Access:** [caatffee.github.io/Caatffee](https://caatffee.github.io/Caatffee/)

---

## 🔐 Signing In

1. Click **"Iniciar sesión"** (Sign in) in the top right corner
2. Discord's authorization page will open
3. Accept the permissions (`identify` and `guilds`)
4. You'll be redirected back to the website, now authenticated

> ⚠️ We only request permissions to **view your profile and servers**. We never access your messages.

---

## 🖥️ My Servers

Once signed in, click your avatar → **"Ver mis servidores"** (My Servers).

You'll see a list of servers where you are the **owner or administrator**.

| Icon | Meaning |
|------|---------|
| 👑 Owner | You own the server |
| 🛡️ Administrator | You have administrator permissions |

Click **"Ir →"** (Go) on any server to open its panel.

---

## 📨 Send Message

From the server panel you can send an embed to any channel.

### How to use it:
1. Select **"📨 Enviar Mensaje"** (Send Message) in the sidebar
2. Choose the destination channel
3. (Optional) Write a title
4. Write your message
5. Pick the embed border color
6. Click **"Enviar con Caatffee"** (Send with Caatffee)

> 💡 The bot must be in the server and have permissions to send messages in that channel.

---

## 🌸 Welcome from the Web

1. Select **"🌸 Bienvenida"** (Welcome) in the sidebar
2. Choose the welcome channel
3. (Optional) Write a custom title and message
4. (Optional) Add an image URL
5. Pick the embed color
6. Click **"✅ Activar"** (Activate)

The configuration is saved in the database and the bot will use it automatically.

---

## 👋 Goodbye from the Web

Same as welcome, but select **"👋 Despedida"** (Goodbye) in the sidebar.

---

## ⚠️ Web Panel Requirements

| Requirement | Description |
|-------------|-------------|
| Bot in server | Caatffee must be added to the server |
| Bot online | The bot must be running |
| Admin permissions | You must be owner or administrator |
| Visible channel | The bot must have access to the channel |

---

## 🛠️ Troubleshooting

| Problem | Possible cause | Solution |
|---------|---------------|---------|
| "Bot not available" | Bot is offline | Wait a few minutes or check hosting |
| "You're not a member of this server" | Token expired | Sign out and sign back in |
| "No administrator permissions" | Lost admin role | Contact the server owner |
| Channel doesn't appear in list | Bot can't see it | Check channel permissions |
| Page doesn't load channels | Connection with bot failed | Reload the page |
