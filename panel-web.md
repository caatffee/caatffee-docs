# 🌐 Panel Web

> 🌍 [English version → Web Panel](../en/web-panel.md)

---

## ¿Qué es el Panel Web?

El panel web de Caatffee te permite gestionar tu servidor **directamente desde el navegador**, sin necesidad de abrir Discord.

🌐 **Acceso:** [caatffee.github.io/Caatffee](https://caatffee.github.io/Caatffee/)

---

## 🔐 Iniciar Sesión

1. Haz clic en **"Iniciar sesión"** en la esquina superior derecha
2. Se abrirá la autorización de Discord
3. Acepta los permisos (`identify` y `guilds`)
4. Serás redirigido de vuelta a la web ya autenticado

> ⚠️ Solo pedimos permisos para **ver tu perfil y tus servidores**. Nunca accedemos a tus mensajes.

---

## 🖥️ Mis Servidores

Una vez iniciada sesión, haz clic en tu avatar → **"Ver mis servidores"**.

Verás una lista de los servidores donde eres **dueño o administrador**.

| Icono | Significado |
|-------|-------------|
| 👑 Dueño/a | Eres el propietario del servidor |
| 🛡️ Administrador/a | Tienes permisos de administrador |

Haz clic en **"Ir →"** en cualquier servidor para abrir su panel.

---

## 📨 Enviar Mensaje

Desde el panel del servidor puedes enviar un embed a cualquier canal.

### Cómo usarlo:
1. Selecciona **"📨 Enviar Mensaje"** en el menú lateral
2. Elige el canal de destino
3. (Opcional) Escribe un título
4. Escribe el mensaje
5. Elige el color del borde del embed
6. Haz clic en **"Enviar con Caatffee"**

> 💡 El bot debe estar en el servidor y tener permisos para enviar mensajes en ese canal.

---

## 🌸 Bienvenida desde la Web

1. Selecciona **"🌸 Bienvenida"** en el menú lateral
2. Elige el canal de bienvenida
3. (Opcional) Escribe un título y mensaje personalizado
4. (Opcional) Añade una URL de imagen
5. Elige el color del embed
6. Haz clic en **"✅ Activar"**

La configuración se guarda en la base de datos y el bot la usará automáticamente.

---

## 👋 Despedida desde la Web

Igual que la bienvenida, pero seleccionando **"👋 Despedida"** en el menú lateral.

---

## ⚠️ Requisitos del Panel Web

| Requisito | Descripción |
|-----------|-------------|
| Bot en el servidor | Caatffee debe estar añadido al servidor |
| Bot online | El bot debe estar activo |
| Permisos de admin | Debes ser dueño o administrador |
| Canal visible | El bot debe tener acceso al canal |

---

## 🛠️ Solución de Problemas

| Problema | Causa posible | Solución |
|----------|--------------|---------|
| "Bot no disponible" | El bot está offline | Espera unos minutos o revisa el hosting |
| "No eres miembro de este servidor" | Token expirado | Cierra sesión y vuelve a iniciar |
| "Sin permisos de administrador" | Perdiste el rol de admin | Contacta al dueño del servidor |
| El canal no aparece en la lista | El bot no puede verlo | Revisa los permisos del canal |
| La página no carga los canales | Conexión con el bot fallida | Recarga la página |
