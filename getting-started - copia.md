## 1. Add the Bot

Click the button below to invite Caatffee to your server:

[![Add Caatffee](https://img.shields.io/badge/➕_Add_Caatffee-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.com/oauth2/authorize?client_id=1475842620063485993&permissions=8&scope=bot%20applications.commands)

> ⚠️ You must be a **server administrator** to add the bot.

---

## 2. Required Permissions

Caatffee requests **Administrator** permission to work correctly.

| Permission | Purpose |
|------------|---------|
| Administrator | Manage channels, roles, members |
| Send Messages | Reply to commands and send embeds |
| Embed Links | Show embeds with images |
| View Channel | Access server channels |
| Manage Server | Edit server information |

> 🔒 We **never** use these permissions for anything beyond what you command.

---

## 3. First Command

Once the bot is added, type in any channel:

```
/help
```

You'll receive a complete guide with all available commands.

---

## 4. Recommended Basic Setup

### 🔔 Set Up Logs
```
/set-logs channel:#your-log-channel
```
Records all server actions in that channel.

### 🌸 Set Up Welcome
```
/set-welcome activate channel:#welcome message:Welcome to {server}, {user}!
```

### 👋 Set Up Goodbye
```
/set-goodbye activate channel:#goodbye message:Goodbye, {user} 👋
```

---

## 5. Web Panel

You can manage your server directly from the official website:

🌐 [caatffee.github.io/Caatffee](https://caatffee.github.io/Caatffee/)

1. Click **"Sign in"** with your Discord account
2. Go to **"My Servers"**
3. Select your server
4. Configure welcome messages, goodbyes, and send messages from there
